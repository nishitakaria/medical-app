This Android application lets its users to send notifications in case of an emergency or a panic situation.
The app which is loaded with several GPS location tracking and emergency contacts. 
This app is capable of sending the emergency messages via push notification and SMS services to the emergency contacts.
The users can send location in the form of text messages on the press of a single button. The
text messages, along with the content, will have the last known location of the user. The
emergency contacts can be set from within the application.
The user can also call saved emergency contacts directly from within the application with a single click, if the nature of the situation demands it.
A fully functional security app, it will be equipped with some of the most astounding features including ambulance booking, location of nearby hospitals, adding your own emergency contacts and also basic first-aid precautionary steps and that might help the users
during an emergency.
